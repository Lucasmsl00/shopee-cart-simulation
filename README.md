# 🛒 Shopee Cart (Simulação de Carrinho de Compras)

Este projeto é uma simulação simples de um carrinho de compras, desenvolvido com **JavaScript (Node.js)**, com foco em prática de lógica, modularização e manipulação de dados.

---

## 🚀 Funcionalidades

* Adicionar produtos ao carrinho
* Remover produtos do carrinho
* Diminuir quantidade de um item
* Listar itens do carrinho
* Calcular o valor total
* Cálculo automático de subtotal por item

---

## 🧠 Conceitos aplicados

* Modularização com ES Modules (`import/export`)
* Uso de funções assíncronas (`async/await`)
* Manipulação de arrays (`push`, `splice`, `reduce`, `findIndex`)
* Factory Function (`createItem`)
* Encapsulamento de lógica no objeto (método `subtotal`)
